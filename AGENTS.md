## Cursor Cloud specific instructions

This repository is a **documentation/portfolio project** for an AI-powered Telegram bot (`@carsavin1_bot`) used in a car selection company. It contains only a `README.md` and screenshot images (`.png` files).

### Key facts

- **No source code exists** in this repository — no programming language files, no dependencies, no build system.
- The actual product is hosted on **Suvvy.ai** (no-code AI assistant platform) and uses **Qwen** (LLM) and the **Telegram Bot API**.
- There are **no tests, linters, build steps, or runnable services** to execute locally.
- The only development actions possible are editing `README.md` and managing image assets.
- `git` is the only tool required.
