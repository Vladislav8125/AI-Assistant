## Cursor Cloud specific instructions

This repository is a **documentation-only showcase** for an AI assistant Telegram bot (`@carsavin1_bot`) for a car selection company. It contains only a `README.md` (in Russian) and screenshot images (`1.png`–`6.png`).

### Key facts

- **No source code**: The bot is built and hosted entirely on external SaaS platforms — [Savvy (suvvy.ai)](https://suvvy.ai/) and Qwen.
- **No dependencies to install**: There is no `package.json`, `requirements.txt`, `Makefile`, `Dockerfile`, or any dependency manifest.
- **No build, lint, or test commands**: Nothing is runnable locally.
- **No services to start**: The Telegram bot runs on the Savvy platform; Google Calendar integration is configured there as well.

### Development workflow

Any changes to this repo are limited to documentation edits (README.md, images). There is no code to lint, test, or build.
