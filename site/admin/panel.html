    Как пользоваться

    1. Добавляйте и редактируйте новости во вкладке «Новости».
    2. Нажмите «Опубликовать на сайте» — изменения появятся на сайте через 1–2 минуты.

    Первый запуск: токен GitHub

    Для публикации нужен Personal Access Token с правом repo. Введите его один раз — он сохранится в браузере.

    Пароль админки

    Задаётся в файле site/data/admin-config.json в репозитории. По умолчанию: andreevsky-admin — смените после первого входа.

    Альтернатива

    Можно также использовать Decap CMS (требует настройки GitHub OAuth).
