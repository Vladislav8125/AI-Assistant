# Админ-панель сайта «Андреевский»

## Быстрый старт

1. Откройте **https://vladislav8125.github.io/AI-Assistant/admin/panel.html**
2. Пароль по умолчанию: `andreevsky-admin` (смените в `site/data/admin-config.json`)
3. Во вкладке **Справка** сохраните [GitHub Token](https://github.com/settings/tokens) с правом `repo`
4. Редактируйте новости и главную страницу
5. Нажмите **«Опубликовать на сайте»** — через 1–2 минуты изменения появятся на сайте

## Что можно редактировать

| Раздел | Что меняется |
|--------|----------------|
| Новости | Заголовок, дата, краткое описание, текст с подзаголовками (Markdown) |
| Главная | Hero-заголовки и цитата внизу страницы |

## Формат новостей на сайте

- **Список** (`/news/`): заголовок → дата → краткое описание → ссылка «Читать полностью»
- **Страница новости**: заголовок → дата → полный текст с оформленными подзаголовками

В редакторе используйте кнопки **H** (заголовок) и **H2** для разделов текста.
